## Descrição

O rocketGames é um e-commerce de consoles, jogos e acessórios. Está sendo desenvolvido como Projeto Integrador, para o curso de Desenvolvimento Web Full Stack da Digital House. 

##  Tecnologias utilizadas:

<li>HTML</li>
<li>CSS</li>
<li>Javascript</li>
<li>Bootstrap</li>
<li>Node.js</li>
<li>EJS</li>
